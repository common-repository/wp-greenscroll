== Description ==

Adds Greenscroll certificate to WP sidebar

== Installation ==

1. Unpack file wp-greenscroll.zip in to wp-content/plugins directory of your Wordpress site.

2a. Go to widget menu and drag and drop the "Greenscroll" widget to your sidebar location

or

2b. Copy and past the below mentioned code to your desired template location

<?php if (function_exists(greenscroll_button)) greenscroll_button(); ?>
